# 🚀 SkillMatch  
A campus-focused platform for mentorship, collaboration & hackathons

---

## 📌 Overview  
SkillMatch is a platform built during a hackathon that enables students to connect with seniors, learn from their experience, and collaborate on hackathons. It creates a campus-driven learning and networking ecosystem.

---

## ✨ Features  
- 👩‍🎓 Connect with seniors for mentorship  
- 🤝 Send & accept connection requests  
- ⚡ Real-time updates using Socket.io  
- 📢 Explore and join hackathon opportunities  
- 🧠 Learn through peer-to-peer interaction  
- 🏫 Campus-focused networking  

---

## 🎯 Problem It Solves  
Students often face:  
- Lack of proper mentorship  
- Difficulty in finding teammates  
- No centralized platform for hackathon collaboration  

---

## 💡 Solution  
SkillMatch provides a dedicated student platform that:  
- Connects juniors with experienced seniors  
- Enables real-time collaboration  
- Simplifies hackathon participation  

---

## 🛠️ Tech Stack  
- **Frontend:** TypeScript  
- **Backend:** Node.js / Express  
- **Database:** MongoDB  
- **Real-time Communication:** Socket.io  

---

## 🚀 How to Run  

```bash
git clone https://github.com/your-username/skillmatch.git
cd skillmatch
npm install
npm run dev
```
